5. Настроить ssh соединение с хостовой ОС на гостевую ОС с использованием ssh ключа и написать команды для:
-- установки ssh сервер
-- добавления порта 22 в исключения файрвола ufw (самостоятельно найти)
-- генерации ssh ключа

В терминал MacOS ввела ssh имя пользователя@ip_адрес(ip a) после чего дальнейшую работу продолжила через терминал MacOS.
Убедившись, что файрволл ufw выключен (командой ufw status) перевела его в активное состояние командой ufw enable. По умолчанию все входящие пакеты отклоняются (ufw default deny incoming), а исходящие допускаются (ufw default allow outgoing).
Для порта 22 введем исключение, добавив правило принимать входящие и исходящие подключения к порту 22 - ufw allow 22.
Ключ генерируется командой ssh-keygen (два ключа: приватный и публичный).
